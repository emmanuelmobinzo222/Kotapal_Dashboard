## First Time? Create an Account

If you don't have an account yet:

### Step 1: Start Both Servers
**Double-click:** `START_BOTH.bat`

Wait for both windows to show:
- Backend: "running on port 3000"
- Frontend: "Compiled successfully"

### Step 2: Open Browser
Go to: **http://localhost:3001**

### Step 3: Create Account
1. Click **"Create a new account"** link
2. Fill in the registration form:
   - **Name:** Your Name
   - **Email:** your@email.com
   - **Password:** (choose a password)
   - **Website:** https://yourwebsite.com (optional)
   - **Plan:** Select a plan (Starter, Pro, Creator+, or Agency)
3. Click **"Create account"** button
4. **You'll automatically be logged in!**

## Troubleshooting

### Can't Log In?

**Check these:**

1. **Backend Running?**
   - Look at "Kota Backend" window
   - Should say "running on port 3000"
   - If not, restart with `START_BOTH.bat`

2. **Frontend Running?**
   - Look at "Kota Frontend" window
   - Should say "Compiled successfully"
   - If not, wait for compilation

3. **Wrong Email/Password?**
   - Make sure you're using the same email/password from registration
   - Try registering a new account

4. **Error Message?**
   - Check browser console (F12)
   - Look for error details
   - Check backend window for error messages

5. **Blank Screen?**
   - Wait a few seconds for loading
   - Refresh the page (Ctrl+R)
   - Clear cache (Ctrl+Shift+Delete)
