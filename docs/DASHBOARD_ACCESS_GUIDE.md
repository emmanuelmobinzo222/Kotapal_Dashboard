## üí° Dashboard Tips

### If Dashboard Looks Empty
1. Look for a **"Show"** button at the top-right
2. Click it to reveal dashboard content
3. Content includes:
   - Statistics cards
   - Analytics charts
   - Recent blocks
   - Performance alerts

### If You Don't See the Sidebar
1. Check if you're on mobile - click the menu icon (‚ò∞)
2. Make sure you're logged in
3. Refresh the page (F5)

### If You Can't Access Pages
1. Make sure you're logged in
2. Check that both servers are running:
   - Backend on port 3000
   - Frontend on port 3001
3. Try logging out and back in
